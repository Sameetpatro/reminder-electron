## ‚ú® Features

### 1. Reminders
- Add reminders with:
  - Task description
  - Deadline
  - Importance flag
- Desktop notifications at:
  - 50% time consumption
  - 90% time consumption
  - 100% (deadline reached)
- Mark reminders as completed or delete them
- Optional email notifications

---

### 2. History
- View completed reminders
- Track cancelled and finished tasks
- Maintain reminder activity history

---

### 3. Technical Skills Tracker
- Add and manage technical skills
- Assign proficiency levels:
  - Beginner
  - Intermediate
  - Advanced
  - Expert
- Useful for self-evaluation and progress tracking

---

### 4. Daily Schedule
- Create and manage daily schedules
- Add time slots with activities
- Edit or update anytime

---

### 5. Attendance Tracker
- Add subjects
- Mark attendance by:
  - Subject
  - Day
  - Date
- View attendance statistics

---

### 6. Email Notifications (Optional)
- Email alerts for reminders
- Supported services:
  - Gmail
  - Outlook
  - Yahoo
- Enable or disable email notifications anytime
- Uses app passwords for security
